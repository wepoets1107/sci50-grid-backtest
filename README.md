## 核心逻辑

### 仓位档位
- **满仓 90%** — 双多头(MA5>MA20+MA10>MA20) + MA5领先MA10
- **重仓 70%** — 双多头 + MA5≤MA10
- **半仓 40%** — 短多长平(MA5>MA20但MA10≤MA20)
- **轻仓 25%** — MA5>MA20 + MA20斜率>0
- **空仓 0%** — MA5≤MA20

### 因子调节
- **布林带**: 上轨×0.80 / 中轨下方×0.90 / 下轨且多头×1.15
- **MA动量**: 偏离MA20 1%/3%/8% 分别加仓 +5%/+10%/+15%
- **过滤**: 初始上限60% | 确认≥1天 | 量比<0.8暂缓 |
- **震荡**: 布林带宽<12%时仓位上限50%
